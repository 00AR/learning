Semantic Elements

Semantic refers to the meaning of a piece of code. For example, <h1> is a semantic element which gives the text its wrap around the role of a 'top level heading on your page.'

Examples of Sematic tags: <article>, <header>, <footer>, <details>, <nav>, <summary>

System Requirements

Requires a computer running an operating system. The computer must have some memory and ideally some kind of long-term storage. An input device as well as some form of output device is recommended.
